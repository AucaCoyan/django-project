# Installation

1. Clone the repo

```
git clone https://github.com/AucaCoyan/django-project
```

2. Install dependencies

```
pipenv install
```

3. Enter into the venv

```
pipenv shell
```

4. Run manage.py

```
cd django_project
python manage.py runserver
```
